1. Переходик в корень диска D:
2. Создаем каталок text
	mkdir test
3. Заходим в каталог
	cd test
4. На github создаем репозиторий test со следующими настройками:
	включаем Public
	включаем Add a README file
	включаем Add .gitignore
		выбираем Java
5. Клонируем репозиторий на локальный диска
	git clone https://github.com/osavenko/test.git
6. Смотрим текущее состояние каталога
	dir
7. Заходим в каталог test и снова смотрим что внутри.
8. Любым удобным способом создать текстовый файл и записать его внутрь текущего каталога.
	я создал lesson.txt с текущим текстом.
9. Повторить п.6

10. Проверяем статус локального репозитория
	git status
11. Выполнить комманду по добавлению текстового файла в список "отслеживания" git
	git add <имя вашего фала>
12. Повторить п. 10.
13. Выполнить commit
	git commit -m 'commit_name'
